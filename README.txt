Voici Pratibook, notre projet de CODEV !
Créateur : Équipe Sympa : Audu Bastien, Giust Alexis et Piscione Thomas

Cette application est là pour aider les bibliothèques et leurs utilisateurs.
Avec cette web-solution, vous pouvez consulter vos œuvres emprunté pour savoir leurs dates de rendus.
Vous pouvez aussi consulter la liste des œuvres de manière simple et intuitive.

Pratibook intègre aussi une gestion de lecture de code barre pour simplifier les emprunts.
